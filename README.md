## Prerequisites

1. Node/NVM
2. Maybe some C/C++ toolchain for sass-related dependencies:sweat_smile:

## Getting Started

```bash
npm install
```

## Develop
```bash
npm run serve
```

## Test Deployment

```bash
npm run serve:dist
```

## Deploy

```bash
npm run build
npm run deploy
```
